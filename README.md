# Linking-Javascript-with-HTML
A **README file** is a text file that explains your project. For your HTML project, create a file named **README.md** in the same folder.

Example:

````markdown
# Simple Button Alert Project

## Description
This is a simple HTML and JavaScript project.

When the user clicks the button, an alert box displaying `100` appears.

## Files

### index.html
Contains the webpage structure and a button.

### lesson1.js
Contains JavaScript code.

Example:

```javascript
console.log(100);
```

## How to Run

1. Open `index.html` in a web browser.
2. Click the **Click!** button.
3. An alert box showing `100` will appear.
4. Open Developer Tools (F12) → Console to see output from `lesson1.js`.

## Author

Muhammad Suffiyan Rafi
````

Save this as **README.md** and it will serve as documentation for your project.
